# Gitnote v1.0.2 — 首个正式发布版本

> **Gitnote** 是一款基于 Git 的本地 Markdown 笔记应用，让知识管理回归本地、回归纯净。

---

## 项目介绍

Gitnote 致力于为用户提供一个**简洁、高效、可同步**的 Markdown 笔记体验。所有数据存储在本地文件系统中，以纯文本 Markdown 格式保存，配合 GitHub 同步功能，实现跨设备无缝访问。

### 核心理念

- **本地优先**：笔记数据完全存储在本地，你的知识你做主
- **纯 Markdown**：无专有格式锁定，笔记即文件，随时可用任何编辑器打开
- **Git 同步**：通过 GitHub 仓库实现多设备同步，版本历史自动保留
- **隐私安全**：所有数据不经第三方服务器，Token 仅用于 GitHub API 交互

---

## 功能特性

### 笔记管理
- 文件夹分类组织笔记，支持自定义 Emoji 标识
- Markdown 实时编辑与预览
- 笔记标签系统，便于检索与归类
- 拖拽排序，灵活调整笔记顺序

### 文件存储
- 每篇笔记以独立文件夹存储，包含 `config.json`（元数据）+ Markdown 文件
- 支持图片内联（base64）和独立文件两种存储模式
- 支持自定义存储路径，灵活管理笔记位置

### GitHub 同步
- 支持单篇笔记同步和批量全量同步
- 智能冲突检测，基于时间戳判断版本新旧
- 支持图片导出模式切换（base64 / 文件）
- 同步状态可视化反馈

### 导出功能
- 将笔记导出到系统文档目录
- 支持图片内联导出和独立文件导出两种模式
- 导出笔记包含完整的元数据配置

### 多语言支持
- 中文 / English 双语界面
- 欢迎笔记内容随语言设置自动切换

### 主题系统
- 亮色 / 暗色 / 跟随系统 三种主题模式
- 基于 Tailwind CSS 的统一设计语言

### 跨平台
- **Windows**：x64 + ia32 安装包
- **macOS**：x64 + arm64（Intel / Apple Silicon）

## 技术架构

| 层级 | 技术选型 |
|------|---------|
| 前端框架 | React 18 + TypeScript |
| UI 样式 | Tailwind CSS |
| 状态管理 | Zustand |
| Markdown 渲染 | Unified + Remark |
| 桌面框架 | Electron 35 |
| 构建工具 | Vite 6 |
| 打包发布 | electron-builder |

| macOS (Apple Silicon) | `Gitnote-*-arm64.dmg` | arm64 |
